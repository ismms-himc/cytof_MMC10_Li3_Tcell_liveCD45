An HCC lesion was surgically resected from the right liver lobe of a 62-year old patient following 2 doses of nivolumab treatment. Cells were dissociated from the tumor, border and adjacent normal tissue and analyzed by CyTOF. Above is an interactive clustergram/heatmap visualization of Louvain/Phenograph clustering results at level 3. Rows and columns have been hierarchically clustered (using SciPy) using cosine distance and average linkage (see Jupyter notebook for more information). Lovain-3 clusters are shown as columns and markers are shown as rows. Louvain clusters are found for three tissue types (see column categories) and the fraction/frequency of all cells within a tissue is shown as the third column value-based category.

Clicking marker/rows brings up t-SNE scatterplots (see below) with overlayed marker levels. See Getting Started for more information on using the interactive Clustergrammer heatmap. See Jupyter notebook for details on data processing.
